# pCreate  – 从文本树快速生成项目结构

一个用 Go 编写的轻量级命令行工具，能根据人类易读的树状结构文本文件，自动创建完整的项目目录和空文件。跨平台兼容 Windows、Linux 和 macOS，非常适合用于脚手架搭建、教学示例或团队统一项目骨架。

## 这个工具解决什么问题

你有个创意，和AI一顿讨论，然后AI丢给你一个项目结构配置和一堆代码。这个项目结构详细给出了项目目录结构和文件分布。

然而，首先你要根据项目结构配置建立这个目录框架，然后再建立代码文件，一个一个手工建立，是不是很烦。

这个工具就是解决这个问题的。

```text
my-app/
├── cmd/server/main.go                # 程序入口
├── internal/
│   ├── config/config.go              # 配置加载
│   ├── db/db.go                      # 数据库初始化
│   ├── handler/
│   │   ├── estimate.go
│   │   ├── export.go
│   │   └── param.go
│   └── middleware/cors.go
├── web/
│   ├── index.html                    # 主页
│   ├── style.css
│   └── app.js
├── go.mod
└── go.sum
```

## 功能特点

- 📁 **自动解析层级**：基于缩进和树状符号正确识别目录嵌套关系
- 📄 **智能区分文件/目录**：自动判断节点是文件还是文件夹，支持无扩展名文件和隐藏目录，如果是文件则创建空文件占位
- 🌐 **跨平台**：内部使用 `filepath` 处理路径，自动适配当前操作系统的路径分隔符
- 📂 **就地生成**：项目结构创建在配置文件所在的目录，干净且直观
- ⚡ **快速执行**：整个配置文件一次性读入解析，适合小型到中型项目

## 安装

确保已安装 Go 1.25.3 或更高版本，然后：

```bash
git clone https://github.com/yourusername/structforge.git
cd structforge
go build -o structforge main.go
```

又或者直接下载 [可执行文件](https://github.com/DavidShiang/pCreate/releases)

## 使用

- 直接终端运行
- 或者把项目结构配置文本文件拖到这个程序图标上


## 配置文件规则说明

- 第一行通常为根目录名，后面不加 `/` 也可以
- 缩进和树状符号（`│`、`├──`、`└──` 等）用于表示层级关系
- 文件名中包含 `.` 的会被识别为文件（如 `main.go`），否则为目录
- 无扩展名但确为文件的名称（如 `Dockerfile`、`Makefile`）需要在名称末尾添加 `# file` 注释来提示（当前版本基于名称规则自动判断，绝大多数情况无需额外处理）
- 隐藏目录（如 `.git`、`.vscode`）会被自动识别为目录
